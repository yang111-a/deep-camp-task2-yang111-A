# deep-camp-task2-yang111-A

## 项目简介

本项目为 **Task 2：AI 辅助四级备考——词汇学习场景**，使用 Claude AI 对四级真题阅读 Passage（Digital Nomad）进行生词提取、记忆卡生成与结构化输出，探索 AI 在语言学习中的实际应用。

## 项目结构

```
deep-camp-task2-yang111-A/
├── README.md
├── task2-ai-scenario/
│   ├── scenario_description.md   # 场景描述：四级备考 + AI 辅助词汇学习
│   ├── task_goal.md              # 任务目标与交付物
│   ├── ai_workflow.md            # AI 交互工作流记录
│   ├── prompts.md                # 用户原始提示词汇编
│   ├── comparison.md             # 不同输出形式对比分析
│   └── reflection.md             # 反思与总结
├── outputs/
│   └── vocabulary_card.md        # 最终交付：规范 Markdown 记忆卡
├── sources/
│   └── references.md             # 真题原文及参考来源
└── screenshots/                  # 截图（如有）
```

## 技术栈

- 内容生成：Claude AI (DeepSeek V4 Pro)
- 输出格式：Markdown
- 版本控制：Git + GitHub
