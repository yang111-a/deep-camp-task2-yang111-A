# 四级真题生词记忆卡——Passage 1（数字游民篇）

## 使用说明

本文档面向四级备考者，精选 Passage 1（Digital Nomad）中的核心生词，逐词拆解记忆。建议每天早晚各刷一轮，先遮住释义看英文回忆，再反向看中文拼写。每个词读例句出声 3 遍，搭配短语一并牢记。坚持"少量多次"原则，每次不超过 10 分钟。

---

### 1. nomad

| 字段 | 内容 |
|---|---|
| 词性 | n. |
| 中文释义 | 游牧者；漂泊者 |
| 真题原句 | The concept of "digital nomad" has gained popularity in recent years. |
| 词根 / 记忆技巧 | nom-（牧场、游牧）源自希腊语 *nomas*；联想：**no + mad** → 游牧者四海为家，不愤怒（不困于一地） |
| 常见搭配 | `digital nomad` 数字游民；`nomadic lifestyle` 游牧式生活 |
| 难度 | ⭐⭐ 需掌握用法 |

---

### 2. remotely

| 字段 | 内容 |
|---|---|
| 词性 | adv.（形容词：remote） |
| 中文释义 | 远程地；轻微地 |
| 真题原句 | ...use technology to work remotely... |
| 词根 / 记忆技巧 | `re-`（向后）+ `mot-`（移动，同 motion）→ 向远处移动 → 远程 |
| 常见搭配 | `work remotely` 远程办公；`not remotely interested` 丝毫不感兴趣 |
| 难度 | ⭐⭐ 需掌握用法 |

---

### 3. flexibility

| 字段 | 内容 |
|---|---|
| 词性 | n.（形容词：flexible） |
| 中文释义 | 灵活性；弹性；柔韧性 |
| 真题原句 | This lifestyle offers flexibility and freedom... |
| 词根 / 记忆技巧 | `flex-`（弯曲）+ `-ibility`（能力名词后缀）→ 能弯曲的能力 → 灵活性。联想：flex 本身表"弯曲肌肉"，flexible = 可弯曲的 → 灵活的 |
| 常见搭配 | `offer flexibility` 提供灵活性；`flexibility in working hours` 工作时间弹性 |
| 难度 | ⭐⭐ 需掌握用法 |

---

### 4. drawn to

| 字段 | 内容 |
|---|---|
| 词性 | phr.（原形动词：draw；过去分词：drawn） |
| 中文释义 | 被……吸引 |
| 真题原句 | ...many young professionals are drawn to this way of life... |
| 词根 / 记忆技巧 | draw 本义"拉"，引申为"吸引"；drawn 是过去分词 → "被拉过去" → 被吸引。联想：磁铁把人 draw 过去，人即被 drawn to 某物 |
| 常见搭配 | `be drawn to sth` 被某物吸引；`draw one's attention` 引起某人注意 |
| 难度 | ⭐⭐ 需掌握用法 |

## 下一步复习计划

1. **当天默写**：遮住释义栏，看英文单词写出中文释义和至少一个搭配，错词做标记。
2. **次日回测**：只复习前一天标记的错词，正确 3 次后取消标记。
3. **第 3 天造句**：每个词造一个与自身经历相关的句子（不要抄例句），加深情景记忆。
4. **一周复盘**：将 4 个词混入 Passage 2 的新词中随机抽测，检验长期记忆。
5. **真题回读**：重读 Passage 1 原文，确认每个词在上下文中能秒反应，无障碍理解全段。
