# 任务目标与交付物

## 总目标

以一篇四级阅读真题为素材，借助 AI 完成从"原文理解 → 生词筛选 → 记忆卡生成 → 格式化输出"的完整学习流水线。

## 子任务拆解

| 序号 | 任务 | 描述 | 交付物 |
|------|------|------|--------|
| 1 | 通读理解 | 阅读 Passage 1（Digital Nomad），概括大意 | 口头总结 |
| 2 | 词汇筛选 | 从原文中筛选 15-20 个四级核心生词 | 词汇表格（19 词） |
| 3 | 进阶筛选 | 用户自选 4 个重点词 | 人工筛选清单 |
| 4 | 记忆卡生成 | 对 4 个重点词逐词生成完整记忆卡 | 桌面 .txt 文件 |
| 5 | Markdown 规范化 | 整合为规范 Markdown 文档，含使用说明与复习计划 | `outputs/vocabulary_card.md` |
| 6 | 项目结构搭建 | 建立完整项目目录，填入所有内容，提交 GitHub | 本项目仓库 |

## 记忆卡字段标准

每张记忆卡包含以下字段：

| 字段 | 说明 |
|------|------|
| 单词 + 词性 | 英文词形及词性标注 |
| 中文释义 | 准确的中文翻译 |
| 真题原句 | 原文中包含该词的句子（无则补四级难度例句） |
| 词根拆解/记忆技巧 | 词源、词根、联想方法（不确定则写"无"） |
| 常见搭配 | 1-2 个高频短语或固定搭配 |
| 难度评级 | ⭐ 认知即可 / ⭐⭐ 需掌握用法 |

## 成功标准

- 所有筛选出的词均符合"四级难度"定位（不过易、不过偏）
- 记忆卡内容可独立使用，脱离原文也能复习
- Markdown 输出结构清晰，可直接用 Typora / VS Code 渲染
- 项目目录完整，文件组织合理
